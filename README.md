# shopping_mall

## 1. 개요
스프링 프레임워크를 사용한 프로젝트입니다. 쇼핑몰의 기본적인 기능들을 MySQL을 연동하여 구현했습니다.

## 2. 개발 환경 및 사용 기술
**Front-End**
- Thymeleaf
- HTML, CSS, JS
- Bootstrap  


**Back-End**
- Spring Boot
- JPA
- MySQL
